
# Book App

Простой Angular-приложение для управления книгами с возможностью добавления, фильтрации и сохранения в localStorage.

## Установка

```bash
npm install
```

## Запуск приложения

```bash
npm run start
```

Откроется приложение на http://localhost:4200

## Тесты

```bash
npm run test
```

Запуск юнит-тестов через Karma и Jasmine.
